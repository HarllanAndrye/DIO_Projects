Projeto com:
- Java 8;
- Maven;
- Spring Boot 2.5.2;
- Banco de dados H2.


### :computer: Rodar a aplicação
---

Para executar o projeto no terminal, digite o seguinte comando:

```shell script
$ mvn spring-boot:run
```

Após executar o comando acima, o seguinte endereço da API estará disponível:

```
http://localhost:8082/api/v1/rooms
```
